# Packaging

Source manifests for Windows package managers.

- `packaging/scoop/hop.json`
- `packaging/winget/elev1e1nSure.hop.yaml`
- `packaging/winget/elev1e1nSure.hop.locale.en-US.yaml`
- `packaging/winget/elev1e1nSure.hop.installer.yaml`

These files point at the Windows release asset published from GitHub Releases.
Update the version, URL, and SHA256 when a new release is cut.
