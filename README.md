# sshm

TUI SSH-менеджер. Управляет `~/.ssh/config` — просмотр, поиск, добавление, редактирование, удаление хостов и подключение к ним. Написан на Go с использованием Bubble Tea.

## Возможности

- Просмотр и фильтрация хостов из `~/.ssh/config`
- Добавление, редактирование и удаление записей
- Прямое SSH-подключение через алиас (все директивы из конфига применяются — ProxyJump, StrictHostKeyChecking и т.д.)
- Индикация доступности хостов (TCP-check с ограничением конкурентности, хосты за ProxyJump/ProxyCommand не сканируются)
- История подключений (JSON)
- Локализация: английский, русский (автоопределение по локали)

## Структура

- `cmd/sshm/` — точка входа CLI.
- `internal/app/` — жизненный цикл приложения.
- `internal/cli/` — разбор аргументов командной строки.
- `internal/config/` — системные пути приложения.
- `internal/sshconfig/` — чтение, разбор и изменение `~/.ssh/config`.
- `internal/sshclient/` — поиск и запуск `ssh`.
- `internal/history/` — хранение истории подключений.
- `internal/i18n/` — русская и английская локализация.
- `internal/ui/` — Bubble Tea TUI.
- `internal/domain/` — доменные модели.
- `internal/util/` — общие файловые и текстовые функции.
- `internal/apperr/` — типизированные ошибки с обёрткой.

## Запуск

```bash
go run ./cmd/sshm
go run ./cmd/sshm --language ru
go run ./cmd/sshm --language en
```

Без `--language` язык определяется по `LC_ALL`, `LC_MESSAGES` или `LANG`; при неизвестной локали используется английский.

## Безопасность

- Все изменения конфигурации атомарны (запись во временный файл + rename).
- Символьные ссылки на `~/.ssh/config` сохраняются при записи.
- Отображаемые значения из SSH-конфига очищаются от управляющих последовательностей терминала.
- Конфигурация перечитывается с диска перед каждой операцией записи для защиты от параллельных изменений.
- При ошибке записи состояние в памяти откатывается.
